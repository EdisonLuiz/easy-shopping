  <strong>Easy Shopping</strong> é uma interface moderna e totalmente responsiva de uma landing page para uma plataforma de compras online, projetada para oferecer uma experiência de navegação fluida, limpa e intuitiva para o usuário.

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

* **HTML5** – Estruturação semântica do conteúdo.
* **CSS3** – Estilização, layouts modernos (Flexbox/Grid) e animações.
* **Design Responsivo** – Adaptação completa para dispositivos móveis, tablets e desktops (Media Queries).
* *(Opcional: Se tiver JavaScript no seu projeto, descomente a linha abaixo)*
    ---

## 🎨 Funcionalidades do Projeto

* [x] **Design Responsivo:** Visualização otimizada para qualquer tamanho de tela (Mobile First / Desktop).
* [x] **Interface Moderna:** Uso de paleta de cores harmoniosa, tipografia limpa e espaçamentos bem definidos.
* [x] **Seções Estruturadas:** Banner principal (Hero), seção de vantagens/produtos em destaque e rodapé institucional.
* [x] **Elementos Interativos:** Efeitos de *hover* em botões e cards de produtos para melhorar a experiência do usuário (UX).
